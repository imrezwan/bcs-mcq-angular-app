# BCS MCQ App
This frontend app consumed a restful API from [BCS MCQ App API](https://github.com/rezwan2525/bcs-mcq-app-api)
The purpose of this project was actually creating a MCQ exam system of BCS model questions.

## Technology Used
* Angular
* Typesript
